Markdown Teacher Planner

Open source · For teachers

A distraction-free planning tool built for teachers. Write your lessons, mark your days, and keep notes in plain Markdown — no accounts, no cloud, no faff.

Try it in browser View on GitHub ↗

Features

Markdown-first

Write lesson notes in plain Markdown. Headings, bold, lists, code blocks — the full format.

Calendar view

Monthly calendar with indicators for days that have entries. Navigate by month, jump to today.

Stays in your browser

All data saved locally — no account, no cloud, no tracking.

Dark mode

Follows your system preference automatically.

Export & import

Export all your data as JSON for backup, or import onto a new device.
