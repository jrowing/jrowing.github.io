## Using Dataloggers in Physics Practicals

This resource shows that **Markdown files work too** — you don't have to write HTML.
Just save a `.md` file with the front matter and Jekyll converts it automatically.

### Equipment checklist

- Datalogger unit (Pasco, Vernier, or similar)
- Appropriate sensor (force, motion, temperature, voltage…)
- USB cable or Bluetooth adapter
- Logger software installed on lab computers

### Tips for good data

1. Always zero/calibrate before recording
2. Set your sample rate *before* you start the experiment — you can't change it mid-run
3. Export as CSV for analysis in Python or a spreadsheet
